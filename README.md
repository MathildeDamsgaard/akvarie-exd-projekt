# Fælles
Vi koblede Mathildes skærm på TV og startede fælles med at kode vores akvarie/startskærm. Her brugte vi opgaven "JS Road Show Animations With CSS and Data" som hjælp.

# Mathilde
Mathilde har haft ansvaret for at få fiskene til at bevæge sig. Hun har også haft ansvaret for vores "game" med udgangspunkt i opgaven "PacMan". Hun har også lavet luk/kryds knappen på info pop-up cards.

# Mikkeline
Mikkeline har haft ansvaret for baggrundsgradienten og sound på fiskene. Hun har også lavet home knappen i vores "game". Hun har også kodet info pop-up cards på alle fiskene. 

# Yulia
Yulia har haft ansvaret for overskriften på startskærmen og 'cursor pointer' property i CSS.

# Cecilie
Cecilie har haft ansvaret for baggrundsbilledet og info pop-up bokse.
